## 1. Introducción al panel administrativo

### 1.1 ¿Para qué sirve el panel?

El panel permite actualizar la información que alimenta el sitio público de El Átomo Inox: productos, categorías, datos de contacto, sucursales y medios de trabajos realizados.

[CAPTURA: Vista general del panel administrativo después de iniciar sesión]

### 1.2 Recomendación antes de hacer cambios

Antes de guardar cualquier cambio, revisa que la información sea clara, que las imágenes correspondan al producto o sucursal correcta, y que los elementos marcados como visibles estén completos.

Resultado esperado: el sitio público muestra información consistente y fácil de entender para los clientes.

Errores comunes:

- Activar productos sin imagen.
- Usar categorías que no funcionan en el catálogo público.
- Escribir teléfonos o enlaces incompletos.
- Subir imágenes poco claras o que no representan el contenido.

## 2. Inicio / Dashboard

### 2.1 ¿Para qué sirve esta sección?

Inicio muestra una lectura rápida del estado general del panel. Sirve para detectar contenido listo, pendiente o incompleto sin revisar cada sección una por una.

[CAPTURA: Dashboard completo, señalando las tarjetas de métricas y el bloque de atención inmediata]

### 2.2 Revisar productos publicables

Objetivo: saber cuántos productos tienen la información mínima para mostrarse correctamente.

Cuándo usarlo: al comenzar una sesión de mantenimiento o después de cargar varios productos.

Pasos:

1. Entra a Inicio.
2. Revisa la tarjeta "Productos publicables".
3. Si el número no coincide con el total, entra a Productos para corregir los pendientes.

Resultado esperado: identificas si hay fichas incompletas antes de que afecten el catálogo público.

[CAPTURA: Tarjeta "Productos publicables" señalada]

### 2.3 Revisar imágenes, sucursales y medios

Objetivo: detectar faltantes visuales o de disponibilidad.

Cuándo usarlo: cuando se agregan productos, sucursales o medios nuevos.

Pasos:

1. Revisa las tarjetas de imágenes, sucursales activas y medios activos.
2. Entra a la tarjeta que indique pendientes.
3. Completa la información faltante en la sección correspondiente.

Resultado esperado: el dashboard deja de mostrar advertencias críticas.

[CAPTURA: Tarjetas de imágenes, sucursales y medios activos señaladas]

### 2.4 Usar Atención inmediata

Objetivo: corregir primero los problemas más importantes.

Cuándo usarlo: cuando el panel muestra estado "Crítico" o "Revisar".

Pasos:

1. Lee cada alerta.
2. Da clic en la alerta.
3. Corrige el dato en la sección a la que te envía.

Resultado esperado: se reducen o desaparecen las alertas del dashboard.

[CAPTURA: Bloque "Atención inmediata" con una alerta señalada]

## 3. Productos

### 3.1 ¿Para qué sirve esta sección?

Productos administra las fichas del catálogo público. Cada ficha puede tener nombre, código o modelo, categoría, precio, material, medidas, descripción, observaciones, imagen, visibilidad, usos recomendados y sucursales relacionadas.

[CAPTURA: Tabla de productos completa, señalando métricas superiores, filtros y columna de acción]

### 3.2 Ver la lista de productos

Objetivo: revisar el estado de todas las fichas del catálogo.

Cuándo usarlo: para auditar qué productos están visibles, ocultos o necesitan revisión.

Pasos:

1. Entra a Productos.
2. Revisa las tarjetas "Visibles en web", "Borradores u ocultos" y "Necesitan revisión".
3. Lee la tabla: foto, producto, categoría, precio, estado, fotos, sucursales, revisión y acción.

Resultado esperado: identificas qué productos están listos y cuáles necesitan corrección.

[CAPTURA: Tabla de productos señalando las columnas Estado, Revisión y Acción]

### 3.3 Filtrar productos

Objetivo: encontrar productos por criterios específicos.

Cuándo usarlo: cuando necesitas revisar solo una parte del catálogo.

Pasos:

1. Presiona "Filtros".
2. Elige uno o varios criterios: material, visibilidad, categoría, precio, sucursales o solo elementos a revisión.
3. Presiona "Aplicar filtros".
4. Usa "Limpiar" para volver a ver todo.

Resultado esperado: la tabla muestra solo los productos que cumplen los criterios.

Errores comunes:

- Olvidar limpiar filtros y pensar que faltan productos.
- Usar precio mínimo mayor al precio máximo.
- Filtrar por sucursal cuando el producto todavía no tiene disponibilidad asignada.

[CAPTURA: Panel de filtros abierto, señalando material, visibilidad, precio y botón "Aplicar filtros"]

### 3.4 Ordenar productos

Objetivo: cambiar el orden visual de la tabla para revisar más rápido.

Cuándo usarlo: cuando buscas por nombre o quieres comparar precios.

Pasos:

1. Da clic en el encabezado "Producto" para ordenar A-Z o Z-A.
2. Da clic en el encabezado "Precio" para ordenar de menor a mayor o de mayor a menor.

Resultado esperado: la tabla cambia de orden sin modificar los productos guardados.

[CAPTURA: Encabezado "Producto" o "Precio" con el icono de orden señalado]

### 3.5 Crear un nuevo producto

Objetivo: registrar un producto nuevo dentro del catálogo.

Cuándo usarlo: cuando El Átomo Inox quiera agregar un producto que todavía no aparece en el catálogo público.

Pasos:

1. Entra a Productos.
2. Presiona "Nuevo producto".
3. Si el panel pide preparar categorías, presiona "Preparar categorías compatibles".
4. Llena el formulario.
5. Agrega una imagen del producto.
6. Revisa que la categoría sea compatible.
7. Presiona "Crear producto".

Resultado esperado: el panel crea el producto y te envía a su pantalla de edición. Los productos nuevos se crean como borrador primero y después se publican si cumplen las condiciones necesarias.

Errores comunes:

- Intentar crear sin imagen.
- Dejar vacío el nombre, código/modelo o categoría.
- Escribir el precio con comas, espacios o símbolos.
- Elegir una categoría marcada como "requiere ajuste".

[CAPTURA: Botón "Nuevo producto" señalado]
[CAPTURA: Formulario de nuevo producto, señalando imagen, nombre, código/modelo, categoría y botón "Crear producto"]

### 3.6 Editar un producto existente

Objetivo: actualizar una ficha ya creada.

Cuándo usarlo: cuando hay que corregir información, cambiar imagen, ajustar precio o completar datos.

Pasos:

1. En Productos, busca el producto.
2. Presiona "Editar".
3. Modifica los campos necesarios.
4. Revisa el checklist de publicación.
5. Presiona "Guardar cambios".

Resultado esperado: el panel guarda los cambios. Pueden tardar hasta 5 minutos en verse en la web pública.

[CAPTURA: Botón "Editar" de un producto señalado]
[CAPTURA: Pantalla de edición con checklist de publicación señalado]

### 3.7 Activar u ocultar un producto

Objetivo: controlar si un producto aparece o no en el catálogo público.

Cuándo usarlo: para publicar productos completos o retirar temporalmente productos que no deben verse.

Pasos:

1. En Productos, ubica la columna Acción.
2. Presiona "Activar" si el producto está oculto, o "Ocultar" si está visible.
3. Si el producto no es publicable, corrige primero los pendientes indicados.

Resultado esperado: el estado cambia a "Visible" u "Oculto".

Advertencia: el panel no permite activar un producto si faltan datos obligatorios de publicación.

[CAPTURA: Botones "Activar" y "Ocultar" señalados en la tabla]

### 3.8 Eliminar un producto

Objetivo: quitar del catálogo un producto que ya no debe existir en el panel.

Cuándo usarlo: solo cuando el producto no se debe conservar ni como borrador.

Pasos:

1. En la fila del producto, presiona el botón de eliminar.
2. Lee la confirmación.
3. Presiona "Eliminar producto".

Resultado esperado: se elimina el producto junto con sus fotos, usos recomendados y relaciones con sucursales.

Advertencia: esta acción no se puede deshacer desde el panel.

[CAPTURA: Modal de confirmación "Eliminar producto" señalado]

### 3.9 Formulario de producto explicado campo por campo

#### Campo: Imagen del producto

Qué es: foto principal que representa el producto en el catálogo.  
Obligatorio: sí al crear un producto.  
Cómo llenarlo: presiona "Añadir imagen" y selecciona una imagen clara del producto.  
Ejemplo correcto: foto nítida del carro, mesa, campana o equipo completo.  
Errores comunes: subir imágenes borrosas, recortadas o de otro producto.  
Notas: la imagen se procesa y guarda como archivo optimizado.

[CAPTURA: Área "Imagen del producto" dentro del formulario señalada]

#### Campo: Nombre del producto

Qué es: nombre visible del producto.  
Obligatorio: sí.  
Cómo llenarlo: usa un nombre corto, claro y descriptivo.  
Ejemplo correcto: Carro taquero grande en acero inoxidable.  
Ejemplo incorrecto: Carro 1.  
Notas: ayuda al cliente a reconocer el producto.

[CAPTURA: Campo "Nombre del producto" señalado]

#### Campo: Código o modelo

Qué es: identificador interno o modelo del producto.  
Obligatorio: sí.  
Cómo llenarlo: escribe el código usado por el negocio.  
Ejemplo correcto: CT-120.  
Errores comunes: repetir códigos o dejarlo como texto genérico.

[CAPTURA: Campo "Código o modelo" señalado]

#### Campo: Categoría

Qué es: grupo del catálogo al que pertenece el producto.  
Obligatorio: sí.  
Cómo llenarlo: elige una categoría activa y compatible.  
Ejemplo correcto: Carros, mesas, campanas o una categoría compatible del panel.  
Errores comunes: seleccionar una categoría marcada como "requiere ajuste".  
Notas: afecta cómo aparece el producto en el catálogo público.

[CAPTURA: Selector "Categoría" y estado de compatibilidad señalados]

#### Campo: Precio

Qué es: precio mostrado o usado como referencia.  
Obligatorio: no.  
Cómo llenarlo: solo números, sin comas, espacios, letras ni signos.  
Ejemplo correcto: 12500.  
Ejemplo incorrecto: $12,500 MXN.  
Notas: si se deja vacío, la web mostrará precio a consultar.

[CAPTURA: Campo "Precio" con texto de ayuda señalado]

#### Campo: Material

Qué es: material principal del producto.  
Obligatorio: no.  
Cómo llenarlo: deja "Acero inoxidable" o elige "Material distinto" para escribir otro.  
Ejemplo correcto: Acero inoxidable.  
Errores comunes: escribir materiales incompletos o mezclados.

[CAPTURA: Selector "Material" y campo "Material distinto" señalado si aplica]

#### Campo: Medidas

Qué es: dimensiones o tamaño del producto.  
Obligatorio: no.  
Cómo llenarlo: usa medidas claras y comparables.  
Ejemplo correcto: 120 x 60 x 90 cm.  
Errores comunes: escribir medidas sin unidad.

[CAPTURA: Campo "Medidas" señalado]

#### Campo: Descripción

Qué es: explicación breve del producto.  
Obligatorio: no.  
Cómo llenarlo: menciona uso, características y beneficios principales.  
Ejemplo correcto: Equipo fabricado en acero inoxidable para uso comercial intensivo.  
Errores comunes: copiar textos demasiado largos o poco claros.

[CAPTURA: Campo "Descripción" señalado]

#### Campo: Observaciones para cotización

Qué es: notas útiles para cotizar o aclarar variantes.  
Obligatorio: no.  
Cómo llenarlo: escribe detalles que el vendedor deba considerar.  
Ejemplo correcto: Disponible con ruedas, puertas o medidas especiales.  
Notas: úsalo para aclaraciones comerciales, no para datos básicos que pertenecen a otros campos.

[CAPTURA: Campo "Observaciones para cotización" señalado]

#### Campo: Visible en web

Qué es: controla si el producto puede mostrarse en el sitio público.  
Obligatorio: no, pero es importante.  
Cómo llenarlo: actívalo solo cuando el producto esté completo.  
Notas: aunque esté marcado, el panel valida que el producto sea publicable.

[CAPTURA: Casilla "Visible en web" señalada]

### 3.10 Usos recomendados

Objetivo: agregar usos que ayuden a explicar para qué sirve el producto.

Cuándo usarlo: cuando un producto puede aplicarse a negocios o contextos específicos.

2. Abre "Usos recomendados".

4. Presiona "Agregar uso".

[CAPTURA: Panel "Usos recomendados" con el campo señalado]

Objetivo: indicar en qué sucursales está disponible o debe mostrarse el producto.

Cuándo usarlo: cuando quieres comunicar disponibilidad específica.

Pasos:

1. Entra a editar un producto.
2. Abre "Sucursales".
3. Presiona la sucursal para agregarla o quitarla.

Resultado esperado: el botón cambia entre "Disponible" y "Agregar".

## 7. Trabajos

### 7.1 ¿Para qué sirve esta sección?

Trabajos administra fotos y videos que aparecen en la sección de trabajos del sitio público. Está dividido en videos de proceso, videos de resultados e imágenes de resultados.

[CAPTURA: Página "Trabajos" completa con los tres carruseles señalados]

### 7.2 Editar videos de proceso

Objetivo: mantener los dos videos principales del proceso.

Cuándo usarlo: cuando se quiere reemplazar uno de los videos de proceso.

Pasos:

1. En Trabajos, entra a una tarjeta de "Videos de proceso".
2. Selecciona el archivo de video.
3. Agrega título, alt text y orden si corresponde.
4. Presiona "Guardar".

Resultado esperado: el video se guarda como WebM y queda activo. Esta sección está limitada a dos videos.

Advertencias:

- El video se comprime a máximo 10MB.
- El video debe durar máximo 60 segundos.
- Los videos de proceso se editan, no se eliminan desde esta pantalla.

[CAPTURA: Carrusel "Videos de proceso" con una tarjeta señalada]
[CAPTURA: Modal de edición de video de proceso señalado]

### 7.3 Agregar o editar videos de resultados

Objetivo: administrar videos que muestran resultados o trabajos terminados.

Cuándo usarlo: cuando se quiere agregar o reemplazar un video de resultado.

Pasos:

1. En "Videos de resultados", presiona "Agregar" o entra a una tarjeta existente.
2. Selecciona el archivo de video.
3. Llena título, alt text, orden y estado activo si aplica.
4. Presiona "Guardar".

Resultado esperado: el video queda disponible en el carrusel de resultados.

[CAPTURA: Botón "Agregar" de videos de resultados señalado]

### 7.4 Agregar o editar imágenes de resultados

Objetivo: administrar fotos de trabajos terminados.

Cuándo usarlo: cuando hay nuevas fotos de proyectos, instalaciones o productos terminados.

Pasos:

1. En "Imágenes de resultados", presiona "Agregar" o entra a una tarjeta existente.
2. Selecciona la imagen.
3. Llena título, alt text, orden y estado activo.

## 8. Buenas prácticas para mantener el catálogo ordenado

- Usa nombres claros y descriptivos.
- Mantén un mismo formato para códigos o modelos.
- Usa imágenes nítidas y relacionadas con el producto correcto.
- Evita publicar productos sin categoría compatible.
- Revisa el dashboard después de hacer cambios importantes.
- Relaciona productos con sucursales cuando la disponibilidad sea relevante.
- Usa el campo de observaciones para detalles de cotización, no para reemplazar la descripción.
- Verifica enlaces de redes sociales, Google Maps y YouTube antes de guardar.

## 9. Problemas comunes y cómo resolverlos

### Un producto no se puede activar

Revisa el checklist de publicación. Normalmente falta imagen, categoría compatible, nombre, código/modelo o algún dato mínimo.

### El precio marca error

### No veo todos los productos

### Los cambios no aparecen en la web pública

### Una sucursal aparece con advertencia

### Un archivo de trabajo no se guarda

## 10. Glosario simple

**Visible en web:** elemento que puede mostrarse en el sitio público.  
**Oculto:** elemento guardado en el panel, pero no visible públicamente.  
**Publicable:** producto que cumple las condiciones para mostrarse correctamente.  
**Categoría compatible:** categoría que el catálogo público reconoce y puede mostrar bien.  
**Alt text:** descripción breve de una imagen o video para accesibilidad y contexto.  
**Orden:** número que define qué elemento aparece primero.  
**WebP:** formato optimizado para imágenes.  
**WebM:** formato optimizado para videos.  
**Storage:** espacio donde se guardan archivos subidos desde el panel.
