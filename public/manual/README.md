# Capturas del manual

Guarda aquí las capturas que deben aparecer en `/dashboard/manual`.

Convención:

```txt
public/manual/<capitulo>/<seccion>-<numero>.webp
```

Ejemplo:

```txt
public/manual/productos/campo-nombre-producto-1.webp
```

Cada placeholder del manual muestra debajo el archivo exacto que espera. Si el archivo no existe, el panel mantiene el bloque de "Captura descriptiva" como pendiente.

Recomendación:

- Usa `.webp`.
- Mantén nombres en minúsculas.
- No uses espacios.
- Exporta capturas en formato horizontal cuando sea posible.
